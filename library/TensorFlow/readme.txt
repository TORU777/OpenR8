Requirements:

1. python (Version 3.5.2 is recommended.)
2. TensorFlow on Windows (command: pip install tensorflow)
3. OpenCV via python (command: pip install opencv-python)
4. Installing TensorFlow object_detection
Please refer to https://github.com/tensorflow/models/blob/master/research/object_detection/g3doc/installation.md

5. Installing TensorFlow dependencies using pip (run as administrator):
pip install pillow
pip install lxml
pip install jupyter
pip install matplotlib

After installing, set environmental variables.
